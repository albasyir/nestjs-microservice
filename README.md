## Run Project

### Development

- install Docker
- setup env for each apps
- make sure port 3000, 3001, 3002, 5100 are free
- run `docker compose up`

#### Port Information

- 3000 is port for order
- 3001 is port for kitchen
- 3002 is port for notification
- 5100 is port for database management
